ШАГ 1: ЗАПУСК BACKEND
────────────────────────────────────────────────────────────────────────────

Откройте PowerShell и выполните:

    cd C:\Users\User\Desktop\эвакуаторы\эвакуаторы\backend
    python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

Backend запустится на: http://localhost:8000
API документация: http://localhost:8000/docs

(Оставьте это окно открытым!)

ШАГ 2: ОТКРЫТЬ ЛЕНДИНГ

    C:\Users\User\Desktop\эвакуаторы\landing\index.html

Или выполните в PowerShell:

    start C:\Users\User\Desktop\эвакуаторы\landing\index.html

Лендинг откроется в браузере!

🧪 ТЕСТИРОВАНИЕ:
════════════════════════════════════════════════════════════════════════════

1. Проверьте Backend:
   Откройте: http://localhost:8000/docs
   Должна открыться Swagger документация

2. Проверьте Лендинг:
   - Нажмите "Рассчитать стоимость"
   - Выберите тип транспорта
   - Проверьте расчет цены
   - Заполните форму контактов

3. Создайте тестового водителя:
   
   Откройте: http://localhost:8000/docs
   
   Найдите: POST /api/drivers/register
   
   Нажмите "Try it out" и вставьте:
   
   {
     "name": "Иван Тестовый",
     "phone_number": "+79991234567",
     "email": "test@driver.ru",
     "license_number": "1234567890",
     "passport_number": "1234 123456"
   }
   
   Нажмите "Execute"
   
   Сохраните "id" из ответа - это ID водителя!

🔑 ОСНОВНЫЕ API ENDPOINTS:
════════════════════════════════════════════════════════════════════════════

ДЛЯ КЛИЕНТОВ:
├── GET  /api/services/                - Список услуг
├── POST /api/orders/                  - Создать заказ
└── GET  /api/orders/user/{phone}      - История заказов

ДЛЯ ВОДИТЕЛЕЙ:
├── POST /api/drivers/register         - Регистрация
├── GET  /api/drivers/{id}/stats       - Статистика
├── GET  /api/drivers/{id}/available-orders - Доступные заказы
└── POST /api/drivers/{id}/accept-order/{order_id} - Принять заказ

Все endpoints доступны в документации: http://localhost:8000/docs

❓ РЕШЕНИЕ ПРОБЛЕМ:
════════════════════════════════════════════════════════════════════════════

BACKEND НЕ ЗАПУСКАЕТСЯ:
→ Проверьте Python версию: python --version (должно быть 3.10+)
→ Переустановите зависимости: pip install -r requirements.txt
→ Проверьте порт 8000: netstat -ano | findstr :8000

ЛЕНДИНГ НЕ ОТКРЫВАЕТСЯ:
→ Попробуйте другой браузер
→ Проверьте путь к файлу
→ Используйте Live Server в VS Code

FLUTTER НЕ РАБОТАЕТ:
→ Проверьте установку: flutter doctor
→ Обновите Flutter: flutter upgrade
→ Проверьте устройства: flutter devices

API НЕ ОТВЕЧАЕТ:
→ Убедитесь что backend запущен
→ Проверьте URL в api_client.dart
→ Для Android эмулятора используйте: http://10.0.2.2:8000
