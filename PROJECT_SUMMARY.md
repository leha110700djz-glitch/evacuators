### 2. ⚙️ Backend API (FastAPI)

**Директория**: `эвакуаторы/backend/`

#### Созданные модули:

##### A. Модели данных (`app/models.py`):
- ✅ **User** - пользователи (клиенты)
- ✅ **Driver** - водители с комиссиями и статистикой
- ✅ **DriverStatus** - статусы водителей (pending, approved, active, blocked)
- ✅ **Order** - заказы с полной информацией
- ✅ **OrderStatus** - статусы заказов (created, searching_driver, driver_assigned, и т.д.)
- ✅ **Service** - типы услуг эвакуации
- ✅ **Vehicle** - транспортные средства водителей
- ✅ **Transaction** - финансовые транзакции
- ✅ **PaymentMethod/PaymentStatus** - способы и статусы оплаты

##### B. API Роутеры:

**`app/routers/drivers.py`** - Полный API для водителей:
```python
POST   /api/drivers/register                    # Регистрация водителя
GET    /api/drivers/{driver_id}                 # Профиль водителя
PUT    /api/drivers/{driver_id}                 # Обновить профиль
POST   /api/drivers/{driver_id}/location        # Обновить геолокацию
POST   /api/drivers/{driver_id}/status          # Онлайн/Оффлайн
GET    /api/drivers/{driver_id}/available-orders # Доступные заказы
POST   /api/drivers/{driver_id}/accept-order/{order_id} # Принять заказ
POST   /api/drivers/{driver_id}/order/{order_id}/status # Обновить статус
POST   /api/drivers/{driver_id}/order/{order_id}/cancel # Отменить заказ
GET    /api/drivers/{driver_id}/stats           # Статистика
GET    /api/drivers/{driver_id}/transactions    # История транзакций
POST   /api/drivers/{driver_id}/withdraw        # Вывод средств
GET    /api/drivers/{driver_id}/active-order    # Текущий заказ
```

**Существующие роутеры**:
- `app/routers/orders.py` - Управление заказами
- `app/routers/products.py` - Услуги (services)
- `app/routers/auth.py` - Авторизация
- `app/routers/categories.py` - Категории (водители)

##### C. Бизнес-логика (`app/services/order_dispatcher.py`):

✅ **Автоматическое распределение заказов**:
```python
- calculate_distance()        # Расчет расстояния (Haversine)
- calculate_driver_score()    # Оценка водителя для заказа
- find_best_driver()          # Поиск оптимального водителя

- notify_nearby_drivers()     # Уведомление водителей
- process_pending_orders()    # Обработка очереди заказов
```

**Алгоритм оценки водителя**:
- 50% - близость к точке подачи
- 30% - рейтинг водителя
- 20% - опыт (количество заказов)

## 🎓 Технические highlights

### Backend:

```python
# Автоматическое распределение заказов
def calculate_driver_score(driver, order, distance_km):
    distance_score = (1 - (distance_km / 30)) * 0.5  # 50%
    rating_score = (driver.rating / 5.0) * 0.3       # 30%
    experience_score = min(driver.completed_orders / 100, 1.0) * 0.2  # 20%
    return distance_score + rating_score + experience_score

# Расчет комиссии
commission = float(order.total_price) * driver.commission_rate
earnings = float(order.total_price) - commission
```
