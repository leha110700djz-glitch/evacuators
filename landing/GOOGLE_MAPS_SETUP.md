# 🗺️ Настройка Google Maps API для CarTaxi

## 📋 Что нужно сделать:

### Шаг 1: Получить API ключ Google Maps

1. **Перейдите в Google Cloud Console:**
   - Откройте: https://console.cloud.google.com/

2. **Создайте новый проект (или выберите существующий):**
   - Нажмите на выпадающий список проектов в верхней части
   - Нажмите "Новый проект"
   - Введите название: "CartTaxi"
   - Нажмите "Создать"

3. **Включите необходимые API:**
   - Перейдите в "API и сервисы" → "Библиотека"
   - Найдите и включите:
     - ✅ **Maps JavaScript API**
     - ✅ **Geocoding API**
     - ✅ **Directions API**
     - ✅ **Places API**

4. **Создайте API ключ:**
   - Перейдите в "API и сервисы" → "Учетные данные"
   - Нажмите "Создать учетные данные" → "API ключ"
   - Скопируйте полученный ключ

5. **Настройте ограничения (рекомендуется):**
   - Нажмите на созданный ключ
   - В разделе "Ограничения приложений":
     - Выберите "HTTP-реферреры"
     - Добавьте ваш домен: `https://yourdomain.com/*`
   - В разделе "Ограничения API":
     - Выберите нужные API (указанные выше)
   - Нажмите "Сохранить"

### Шаг 2: Вставьте ключ в код

Откройте файл `index.html` и найдите строку:

```html
<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&libraries=places&language=ru"></script>
```

Замените `YOUR_API_KEY` на ваш настоящий ключ:

```html
<script src="https://maps.googleapis.com/maps/api/js?key=AIzaSyBxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx&libraries=places&language=ru"></script>
```

### Шаг 3: Тестирование

1. Откройте `index.html` в браузере
2. Перейдите к секции "Заказать эвакуатор"
3. Кликните на карту - должна появиться зеленая точка (откуда)
4. Кликните второй раз - красная точка (куда)
5. Должен отобразиться маршрут и рассчитаться расстояние

## 🎯 Как работает карта:

### Пользовательский сценарий:

1. **Пользователь кликает по карте первый раз**
   - Устанавливается зеленая метка (откуда)
   - Адрес определяется автоматически

2. **Пользователь кликает по карте второй раз**
   - Устанавливается красная метка (куда)
   - Адрес определяется автоматически
   - Строится маршрут между точками
   - Рассчитывается расстояние

3. **Автоматический расчет стоимости**
   - Базовая цена (зависит от типа транспорта)
   - + 40₽ за каждый км свыше 10 км
   - = Итоговая стоимость

4. **Отправка заявки**
   - Заполнение формы
   - Отправка в backend API
   - Заявка распределяется ближайшим водителям

## 💰 Монетизация и комиссия:

### Как работает система комиссий:

1. **Клиент заказывает эвакуатор**
   - Видит полную стоимость (например, 3000₽)

2. **Водитель принимает заказ**
   - Backend автоматически рассчитывает:
     - Комиссия платформы: 3000₽ × 15% = 450₽
     - Заработок водителя: 3000₽ - 450₽ = 2550₽

3. **После выполнения заказа**
   - Клиент оплачивает 3000₽
   - Водителю начисляется 2550₽
   - Платформе - 450₽ (ваша прибыль)

4. **Вывод средств водителем**
   - Водитель может вывести деньги в любое время
   - Минимальная сумма: 1000₽

### API интеграция:


1. **Мобильное приложение**

   - Кнопки: "Принять" / "Отклонить"

3. **SMS уведомления (будущее)**
   - Текстовое сообщение с деталями

### Преимущества для водителей:

## 🔒 Безопасность:

### Не храните API ключ в публичном репозитории!

Если используете Git:
1. Создайте файл `config.js`:
```javascript
const GOOGLE_MAPS_API_KEY = 'your_key_here';
```

2. Добавьте в `.gitignore`:
```
config.js
```

3. В HTML подключите:
```html
<script src="config.js"></script>
<script src="https://maps.googleapis.com/maps/api/js?key=" + GOOGLE_MAPS_API_KEY + "&libraries=places"></script>
```
