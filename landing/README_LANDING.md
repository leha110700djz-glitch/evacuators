## 🚀 Как открыть:

### Способ 1 - Просто открыть:
```
Двойной клик по файлу:
C:\Users\User\Desktop\эвакуаторы\landing\index.html
```

### Способ 2 - Live Server (рекомендуется):
1. Откройте VS Code
2. Установите расширение "Live Server"
3. Правый клик на `index.html` → Open with Live Server
4. Сайт откроется на: http://localhost:5500

### Способ 3 - PowerShell:
```powershell
start C:\Users\User\Desktop\эвакуаторы\landing\index.html
```
