🗺️ КАК РАБОТАЕТ КАРТА:
════════════════════════════════════════════════════════════════════════════

ШАГИ ДЛЯ КЛИЕНТА:

1. Перейдите к секции "Заказать эвакуатор"
2. Кликните по карте 1 раз - появится зеленая метка (откуда)
3. Кликните по карте 2 раз - появится красная метка (куда)
4. Автоматически строится маршрут и рассчитывается расстояние
5. Заполните форму (имя, телефон, тип транспорта)
6. Нажмите "Отправить заявку водителям"

РЕЗУЛЬТАТ:
→ Заявка отправляется в backend
→ Backend находит ближайших водителей
→ Водители получают уведомление
→ Первый принявший получает заказ
→ С оплаты взимается ваша комиссия 15%

2. BACKEND API (должен быть запущен)
   
   cd C:\Users\User\Desktop\эвакуаторы\эвакуаторы\backend
   python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   
   Проверка: http://localhost:8000/docs

3. ТЕСТИРОВАНИЕ
   
   ✅ Откройте landing\index.html
   ✅ Перейдите к секции "Заказать эвакуатор"
   ✅ Попробуйте выбрать точки на карте
   ✅ Заполните форму и отправьте

📱 ДЛЯ ВОДИТЕЛЕЙ:
════════════════════════════════════════════════════════════════════════════

Как водители получают заказы:

1. Заявка создается на сайте
2. Backend API находит ближайших водителей (в радиусе 20 км)
3. Водители получают уведомление в приложении:
   
   📲 Новый заказ!
   📍 От: ул. Ленина, 1
   🏁 До: ул. Пушкина, 10
   📏 Расстояние: 5.2 км
   💰 Ваш заработок: 2550₽ (3000₽ - 15%)
   
   [Принять] [Отклонить]

4. Первый принявший получает заказ
5. После выполнения деньги зачисляются на баланс
