Политика конфиденциальности

Дата публикации: 01 января 2025 г.  |  Версия 1.1  |  г. Москва

Сервис «Везунчик» (далее — «Оператор») уважает приватность пользователей и обязуется защищать их персональные данные. Настоящая политика описывает, какие данные собираются, как используются и как защищаются.

🔒 Кратко о главном: Мы собираем только необходимые данные для исполнения заказа. Мы не продаём ваши данные. Вы можете в любой момент запросить удаление своих данных.

2. Правовое основание обработки

Обработка персональных данных осуществляется на следующих основаниях (ФЗ-152 «О персональных данных»):

  • Согласие субъекта персональных данных (п. 1 ч. 1 ст. 6)
  • Исполнение договора, стороной которого является субъект (п. 5 ч. 1 ст. 6)
  • Осуществление прав и законных интересов оператора (п. 7 ч. 1 ст. 6)

3. Как мы используем данные

  • Исполнение заказа: передача адреса и контакта водителю-исполнителю
  • Связь: перезвон для подтверждения заявки, уведомления об изменениях
  • Telegram-уведомления: заявки направляются диспетчеру через Telegram Bot API
  • Улучшение сервиса: анализ агрегированной статистики использования
  • Бонусная программа: учёт баллов в LocalStorage браузера или профиле приложения

4. Передача данных третьим лицам

Мы передаём ваши данные только в следующих случаях:

  • Водители-исполнители: телефон и адрес подачи для выполнения заказа
  • Telegram (Meta Platforms): заявки через Bot API — только текстовые данные заказа
  • Яндекс (Яндекс.Карты): координаты для отображения маршрута (политика Яндекса применяется)
  • Государственные органы: только по требованию закона

Мы не продаём и не передаём данные рекламным агентствам, агрегаторам, маркетологам.

5. Cookie и LocalStorage

5.1. Что хранится в браузере:

  • cookieAccepted — факт принятия политики cookie
  • vezunBonuses — количество бонусных баллов
  • vezunOrders — история заказов (до 50 записей)
  • evac_admin_chat_id — технический идентификатор для Telegram API

5.2. Для базовой работы сайта достаточно функциональных cookie. Аналитические cookie используются только с вашего согласия.

5.3. Вы можете очистить данные через настройки браузера в любой момент.

6. Защита данных

  • Передача данных — только по HTTPS/TLS
  • Доступ к данным заказов — только у уполномоченных сотрудников
  • Telegram Bot Token хранится только на сервере, не публикуется
  • Мобильное приложение не запрашивает лишних разрешений

8. Ваши права (ФЗ-152)

  • Право на доступ: запросить перечень своих данных
  • Право на исправление: уточнить неверные данные
  • Право на удаление: запросить удаление всех данных
  • Право на отзыв согласия: в любой момент без указания причин
  • Право на жалобу: в Роскомнадзор (rkn.gov.ru)

Для реализации прав обратитесь: Telegram @dobrycode или телефон 8 (977) 262-81-82. Срок ответа — 30 дней.

9. Дети

Сервис не предназначен для лиц моложе 18 лет. Мы сознательно не собираем данные несовершеннолетних. Если вам стало известно о предоставлении данных ребёнком — сообщите нам.

10. Изменения политики

Мы можем обновлять настоящую политику. Актуальная версия всегда доступна на сайте. При существенных изменениях — уведомление на главной странице.
